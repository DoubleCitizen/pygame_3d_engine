# 3D Engine (pygame)

Простой 3D-рендерер на Python с использованием pygame. Реализует перспективную проекцию, управление камерой от первого лица и поддержку 4D-объектов (тессеракт).

## Возможности

- Перспективная проекция 3D-точек на 2D-экран (pinhole camera)
- Управление камерой мышью и клавиатурой (yaw/pitch)
- Painter's algorithm — сортировка объектов по глубине перед отрисовкой
- Вычисление нормалей граней для backface culling
- Поддержка 4D-координат и гипервращений (тессеракт)
- HUD с FPS, позицией и углами камеры

## Объекты

| Класс | Описание |
|---|---|
| `Cube` | Куб с заданным центром и радиусом |
| `Surface` | Плоская поверхность |
| `Tesseract` | Тессеракт — 4D-гиперкуб, проецируемый в 3D |

## Структура проекта

```
├── main.py               # Точка входа, главный цикл
├── camera.py             # Функции проекции (линейная, матричная, scipy)
├── my_math.py            # Утилиты: конвертация углов, расстояние
├── objects/
│   ├── base_objects.py   # Базовый класс 3D-объекта с поддержкой вращения
│   ├── cube.py           # Куб
│   ├── surface.py        # Плоскость
│   ├── camera.py         # Класс Camera (ввод, обновление позиции)
│   └── tesseract.py      # Тессеракт с 4D-вращениями
└── pygame_ext/
    └── renderer_3d.py    # Renderer3D — очередь задач отрисовки
```

## Установка и запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

**Зависимости:** `pygame==2.6.1`, `numpy==2.4.6`, `scipy==1.17.1`
